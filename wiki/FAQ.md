# FAQ

## Can I use this in a commercial product?

Not under the default non-commercial terms. You need a paid commercial license.

## Is TypeScript required?

The package is authored in TypeScript and publishes compiled output.

## Does it mutate the original array?

No. The API returns a new sorted array.

## Where should I wire this in my app?

Use your app's common components layer as an adapter boundary.

## How do I report security issues?

Use repository security advisories or the contact details in SECURITY.md.
